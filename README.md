# Piedra, Papel o Tijera

Juego básico de Piedra, Papel o Tijera hecho en Python usando Tkinter.

## Cómo jugar

1. Ejecuta el archivo juego.py.
2. Haz clic en Piedra, Papel o Tijera.
3. La computadora elige automáticamente.
4. Se muestra quién gana cada ronda.
5. El primero en llegar a 5 puntos gana la partida.

## Requisitos

- Python 3
- Tkinter (incluido en Python)

## Cómo ejecutar

python juego.py
